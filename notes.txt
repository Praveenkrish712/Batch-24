HTML - Hyper Text markup language (HTML5)  <head> content <head/>

Self Closing tag - hr, meta, br, img

==> To create a Web page
==> Not a Programming language
==> It is a markup language
==> Every Website can start with HTML

1) Heading - h1 to h6

2) Paragraph - p

3) Horizontal - hr

4) break tag - br

5) Text Formatting - i, b or strong, sub, sup, u and mark

6) Anchor tag (Linking) - a

7) Image tag - img

8) list - ordered list and Unordered list

9) Tables - tr, th and td
